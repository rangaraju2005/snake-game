

Snake Game in Python

A classic **Snake Game** built using **Python** and **Pygame**.
Control the snake, eat food, grow longer, and try to achieve the highest score without hitting the walls or yourself.


*  Green blocks ‚Üí Snake
*  Red block ‚Üí Food
*  Black background ‚Üí Game area



Features

* Smooth snake movement
* Score tracking system
* Game Over screen with replay option
* Keyboard controls
* Simple and beginner-friendly code

 Controls

 Left Arrow   Move Left  
 Right Arrow  Move Right 
 Up Arrow     Move Up    
 Down Arrow   Move Down  
 C           Play Again 
 Q           quit game



 Concepts Used

* Game Loop
* Event Handling
* Collision Detection
* Lists & Coordinates
* Object Movement
* Score Management
* Modular Functions
